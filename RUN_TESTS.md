# دليل سريع - Quick Guide

## ⚡ البدء السريع (5 دقائق)

### 1. إعداد قاعدة البيانات
```bash
mysql -u root -p -e "DROP DATABASE IF EXISTS homework_std_ro_db;"
mysql -u root -p -e "CREATE DATABASE homework_std_ro_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
mysql -u root -p homework_std_ro_db < database/schema.sql
mysql -u root -p homework_std_ro_db < database/seeder.sql
```

### 2. تشغيل الاختبارات التلقائية
```bash
php tests/automated_tests.php
```

### 3. اختبار يدوي سريع
1. افتح: `http://localhost/std_2_hotel_system/`
2. سجل دخول مستخدم: `ahmed@demo.com` / `password123`
3. سجل دخول مدير: `admin` / `admin123` على `/admin/login`

## 🐛 حل المشاكل السريع

### المشكلة: "Table doesn't exist"
```bash
# الحل: أعد استيراد قاعدة البيانات
mysql -u root -p homework_std_ro_db < database/schema.sql
mysql -u root -p homework_std_ro_db < database/seeder.sql
```

### المشكلة: "Invalid credentials"
```
الحل: تأكد من استخدام البيانات الصحيحة:
- مدير: admin / admin123
- مستخدم: ahmed@demo.com / password123
```

### المشكلة: "Cannot connect to database"
```
الحل:
1. تأكد من تشغيل MySQL في XAMPP
2. تحقق من اسم قاعدة البيانات: homework_std_ro_db
3. تحقق من إعدادات config/database.php
```

### المشكلة: الصفحة فارغة أو خطأ 500
```
الحل:
1. افتح php_error.log
2. تحقق من صلاحيات المجلدات
3. تأكد من تفعيل display_errors في php.ini
```
