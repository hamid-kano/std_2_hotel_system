## 🔧 نصائح الاختبار - Testing Tips

### 1. استخدم بيانات تجريبية
- لا تستخدم بيانات حقيقية أثناء الاختبار
- استخدم البيانات التجريبية المتوفرة في seeder.sql

### 2. اختبر في بيئة منفصلة
- لا تختبر على قاعدة البيانات الإنتاجية
- استخدم قاعدة بيانات منفصلة للاختبار

### 3. وثق كل شيء
- سجل جميع الأخطاء التي تجدها
- التقط لقطات شاشة
- احتفظ بسجل للاختبارات

### 4. اختبر السيناريوهات السلبية
- لا تختبر فقط الحالات الصحيحة
- جرب إدخال بيانات خاطئة
- جرب الوصول بدون صلاحيات

### 5. اختبر على متصفحات مختلفة
- Chrome
- Firefox
- Safari
- Edge
